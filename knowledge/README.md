# Knowledge Base

Add your project documentation, specifications, and reference files here.

## Usage

Place files in this folder that Claude should reference during development:
- API specifications (.json, .yaml)
- Requirements documents (.md, .txt)
- Database schemas (.sql)
- Integration guides (.md)
- Any other reference material

## Supported Formats
- Markdown (.md)
- Text (.txt)
- JSON (.json)
- YAML (.yaml, .yml)
- SQL (.sql)
- Python (.py)
- TypeScript/JavaScript (.ts, .js)

## Integration with Claude

Files in this folder are automatically referenced in CLAUDE.md.
When you add files here, update CLAUDE.md to include them in the Knowledge Base section.
